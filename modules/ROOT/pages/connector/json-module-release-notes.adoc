== 2.0.0

*July 5, 2018*

=== Min Mule Version
*Mule 4.1.0*

=== New Features and Enhancements

Errors are no longer reported as a List of Strings. Errors are now reported as an actual JSON document. For more details please refer to the xref:connectors::json/json-schema-validation.adoc[Validating Documents against a JSON Schema with the JSON Module] article.

== 1.0.1

*May 7, 2018*

=== Min Mule Version
*Mule 4.1.0*

=== New Features and Enhancements

* Input metadata is more accurate. The connector used to communicate that required
any kind of structure. Now, it is more accurate and shows that it requires any *JSON* structure.
* Upgraded Jackson version due to https://nvd.nist.gov/vuln/detail/CVE-2018-7489[this vulnerability]
